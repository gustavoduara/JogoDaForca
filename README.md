# JogoDaForca.ConsoleApp

![](https://i.imgur.com/tFZmGlJ.gif)

## Projeto

Desenvolvido durante o curso da **Academia do Programador**

## Descrição

O programa escolherá uma entre varias palavras pré-definidas e o jogador terá
que adivinhar qual é a palavra, tendo como dica o número de letras da palavra.

## Funcionalidades

- Escolha de uma palavra aleatória

- O jogo ira contabilizar o número de erros, e termina se o máximo de erros for atingido

- As letras corretas serão exibidas na palavra

- a forca é desenhada confore erro do jogador

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.

 ## Como Usar
 
 #### Clone o Repositório
 ```
 git clone https://github.com/gustavoduara/JogoDaForca
 ```
 
 #### Navegue até a pasta raiz da solução
 ```
 CD JogoDaForca
 ```
 
 #### Restaure as dependências
 ```
 dotnet restore
 ```
 
 #### Navegue até a pasta do projeto
 ```
 cd JogoDaForca.ConsoleApp
 ```
 
 #### Execute o projeto
 ```
 dotnet run
 ```
